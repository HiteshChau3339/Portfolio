Projects Here you will find some of the personal and clients projects that I created with each project containing its own case study

Student Management System (SMS)

The Student Management System (SMS) is a web application with separate roles for admins and students. Admins can add students, post notices, approve leave requests, and manage events. Students can view notices, participate in events, and submit leave requests. The backend is built with PHP, using MySQL for data storage, while the frontend is created with HTML, CSS, JavaScript, and templates. This system enhances student management, communication, and event participation.

Restaurant Management System in c#

The Restaurant Management System is a software solution developed using C# and MySQL Server for efficient data management. It features an admin section where the hotel owner can easily manage products, categories, customers, and orders. The system allows the owner to add new products, organize them into categories, track customer details, and manage orders seamlessly. The user-friendly interface ensures smooth operation and quick access to essential information. This system simplifies restaurant management, improving overall efficiency and customer experience..
